# FormControl

Display styled form controls.

## Basic Input Types

The styles of the Form Control is applied and valid for all of the basic input types:

<MultipleCanvas
  components={[
    { title: 'text', of: FormControlStories.Text },
    { title: 'password', of: FormControlStories.Password },
    { title: 'email', of: FormControlStories.Email },
    { title: 'date', of: FormControlStories.Date },
    { title: 'datetime-local', of: FormControlStories.DateTimeLocal },
    { title: 'color', of: FormControlStories.Color },
    { title: 'file', of: FormControlStories.File },
    { title: 'month', of: FormControlStories.Month },
    { title: 'week', of: FormControlStories.Week },
  ]}
  renderWidth={300}
/>

## Color Variants

We provide some semantic color variant of the inputs: default, valid and invalid inputs:

<MultipleCanvas
  components={[
    { title: 'default', of: FormControlStories.Default },
    { title: 'valid', of: FormControlStories.Valid },
    { title: 'invalid', of: FormControlStories.Invalid },
  ]}
  renderWidth={300}
/>

## Help Text
