# Typography

All typography tokens used by the design system.

## Font Family

The font used in the design system is the [Ubuntu font-family](https://fonts.google.com/specimen/Ubuntu).

| Variable               | Value                   |
| ---------------------- | ----------------------- |
| `var(--font-family)`   | `'Ubuntu', sans-serif`  |

## Font Weight

| Variable                     | Name    | Value     |
| ---------------------------- | ------- | --------- |
| `var(--font-weight-light)`   | Light   | 300       |
| `var(--font-weight-regular)` | Regular | 400       |
| `var(--font-weight-medium)`  | Medium  | 500       |
| `var(--font-weight-bold)`    | Bold    | 700       |

## Font Size

The font size tokens are defined in two subsections: the base text, used for common text, and the headings, used for the h1-h6 headings.

### Base Text

The base text font size tokens are used for common locations.
A basic example is the usage of `var(--font-size-small)` at the
<LinkTo kind="Inventare/FormControl" story="docs" >FormControl</LinkTo> helper text, and the `var(--font-size-base)` at the FormControl label.

| Example                                                                         | Variable                 | Name        | Value       |
| ------------------------------------------------------------------------------- | ------------------------ | ----------- | ----------- |
| <span style={{ fontSize: 'var(--font-size-small)' }}>Instituto Inventare</span> | `var(--font-size-small)` | Small Text  | `0.825rem`  |
| <span style={{ fontSize: 'var(--font-size-base)' }}>Instituto Inventare</span>  | `var(--font-size-base)`  | Text Base   | `1rem`      |
