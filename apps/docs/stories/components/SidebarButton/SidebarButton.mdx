# Sidebar Button

The sidebar button component is a set of utility for button to create a complete sidebar navigation menu. A complete example of sidebar button utilites is:

<Story of={SidebarButtonStories.CompleteExample} />

We have some types of Sidebar Button:
- Sidebar Button: used to indicate the sidebar button without subgroup's.
- Dropdown: used to indicate a group of items.
- Child: used to indicate a child item of a group.
