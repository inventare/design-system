# Avatar

Helper classes to define avatar's on pages.

## Text

We have option to show text on the avatar component, like this:

<Canvas of={AvatarStories.TextDefault} />

## Image

Also, is possible to use the avatar component to show a image, like:

<Canvas of={AvatarStories.ImageTag} />

Or using `background-image`:

<Canvas of={AvatarStories.ImageBackground} />
